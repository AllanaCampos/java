Em java e em C++ utiliza-se a seguinte sintaxe na declaração de uma variável:

tipo identificador [= valor];

Onde tipo é o tipo de dado, identificador é o nome da variável e o valor é o que será atribuído à variável, sendo que é opcional inicializar a variável durante sua declaração.

Em java existem oito tipos de dados primitivos mais comuns, são eles:
1- byte;
2- short;
3- int;
4- long;
5- boolean;
6- char;
7- float;
8- double.

Em C++ os tipos de dados primitivos mais utilizados são:
1- int;
2- double;
3- bool;
4- char;
5- wchar_t.
